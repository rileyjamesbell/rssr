RSSR - Riley's Simple Speed Reader

RSSR is a program for system-wide speed reading.

REQUIREMENTS:
sselp (http://tools.suckless.org/x/sselp/)
python3 (for tkread)
python3-tkinter
python3-re
ncurses

INSTALLATION:
Build with build.sh
Install with install.sh

UNINSTALLATION:
Uninstall with "./install.sh uninstall"

RECOMMENDATIONS:
You bind rssr to some key with a program such as sxhkd.

CONFIGURATION:
You can set the default interval between each line output by setting the environment variable "RSSRMS"
You can set the default added interval for each character by setting the environment variable "RSSRLM"
